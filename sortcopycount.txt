function picked from some_list (1. translate, 2. title 3. strip)
Type help and put the three picked functions from the list with some_list in parenthesis and figure out what they do

1. help(some_list.reverse)
Help on built-in function reverse:
reverse() method of builtins.list instance

2. help(some_list.sort)
Help on built-in function sort:

sort(*, key=None, reverse=False) method of builtins.list instance
    Sort the list in ascending order and return None.
    The sort is in-place (i.e. the list itself is modified) and stable (i.e. the
    order of two equal elements is maintained).
    If a key function is given, apply it once to each list item and sort them,
    ascending or descending, according to their function values.
    The reverse flag can be set to sort in descending order.
